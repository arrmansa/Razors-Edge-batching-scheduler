# Simulation Results

All simulations use estimator arrays generated by `demos/scheduler_tests/save_estimators_for_simulation.ipynb`.

## Global simulation configuration

- Seeds: `[42, 43, 44, 45, 46]`
- Concurrent users (queue cap): `16`
- Requests per run: `20000`
- Queue discipline: refill-to-capacity each scheduler tick (saturated regime)
- Baseline: fixed-cap FIFO without sorting, sweeping all supported `n`
- Razor's Edge strategy set: `FIFO`, `MINMAX`, `BATCH_SIZE`

## Mechanism-isolation ablation (fixed n = best baseline n)

Variant definitions:
- A: baseline fixed-cap FIFO, unsorted queue
- B: sort-only fixed-cap (same `n` as A)
- C: sort + DP + FIFO objective
- D: sort + DP + MINMAX objective
- E: sort + DP + BATCH_SIZE objective
- F: sort + greedy batch-construction + MINMAX objective
