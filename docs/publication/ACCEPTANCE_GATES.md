# Publication Acceptance Gates Index

We maintain two acceptance gate versions:

- **V1 (preprint):** notebook-based checks in `docs/publication/ACCEPTANCE_GATES_V1.md`.
- **V2 (post-preprint):** metric-comparison checks in `docs/publication/ACCEPTANCE_GATES_V2.md`.

Use V1 before preprint submission and V2 after preprint for stricter comparator-based validation.
